# 0.16.3

## 変更点

ここでは主要な変更を紹介します。全ての変更点については[CHANGELOG.md]をご覧ください。

### `AudioQuery`/`AccentPhrase`/`Mora`における不正な状態というものが定義され、明示的にエラーになるように

メソッドとして`{AudioQuery,AccentPhrase,Mora}::validate`が追加されます。`Synthesizer`の各メソッドも内部でバリデートを行い、早期に明示的なエラーを出すようになります。以前は不正なAudioQueryはクラッシュを引き起こしていました。

```py
from voicevox_core import InvalidQueryError

# 例: `vowel`に子音
audio_query.accent_phrases[0].moras[0].vowel = "k"

try:
    audio_query.validate()
except InvalidQueryError as e:
    ...
```

一部の状態については警告を出すことに留めていますが、将来的にすべてエラーになる予定です。

### バグ修正

ダウンローダーにおける辞書 (`dict`)のダウンロード元であるjaist.dl.sourceforge.netが[消失して]しまってダウンロード不可になったため、別の場所からダウンロードするようにしました。
